FFT 二选一：默认 FFTW3F，USE_MKL=ON 时用 Intel MKL。 射频后端（UHD/bladeRF/SoapySDR）任选；纯软件仿真用 ZeroMQ 虚拟射频。 ENABLE_TTCN3=ON 时另需 RapidJSON；BUILD_STATIC=ON 支持全静态链接。
