## 演示逻辑递进

```
第一节：知识建设（16min 演示 + 8min PPT）
  演示一 → CLAUDE.md（Layer 1）—— AI 认识项目基础信息
  演示二A → HTML 全景视图（Layer 2 给人看）—— 团队成员理解项目
  演示二B → CODEMAP + LLM Wiki（Layer 2 给 Agent 看）—— AI 理解代码结构
  演示三 → AST 索引 + LSP（Layer 3）—— AI 拥有编译器级搜索
  [PPT] 增量维护 —— 知识保持新鲜

第二节：Skill × MCP（14min 演示 + 20min PPT + 2min 休息）
  [PPT] Skill/MCP 判断标准 + 场景全景 + 四阶段理论
  演示四 → MCP 搭建 + ToolSearch —— 给 AI "手"（连接外部系统 + 渐进式加载）
  演示五 → Skill 开发+测试 —— 给 AI "脑"（分流 SOP + 代码检查）

第三节：迭代闭环（9min 演示 + 8min PPT）
  [PPT] 效益对比 + 完整流程
  演示六 → dev-iterate Skill + 三层守护 —— AI 能自主编译测试修复

第四节：安全防护（5min 演示 + 14min PPT）
  [PPT] CVE 冲击 + 安全知识库 + 架构
  演示七 → 多 Agent 安全审查 —— AI 受控安全运行
  [PPT] PreToolUse Hook 过程安全
```

### 各演示间的串联要点

| 衔接点 | 前一演示 | 后一演示 | 讲师串场 |
|--------|---------|---------|---------|
| Layer 1 → Layer 2（人） | 演示一 CLAUDE.md | 演示二A HTML | "CLAUDE.md 是给 AI 的入口——但团队成员也需要全景理解" |
| Layer 2（人）→ Layer 2（Agent） | 演示二A HTML | 演示二B CODEMAP | "HTML 给人看全局——CODEMAP 给 Agent 看精确位置" |
| Layer 2 → Layer 3 | 演示二B CODEMAP | 演示三 AST | "CODEMAP 是知识层——AST 是编译器层，两者互补" |
| 知识 → 工具 | 演示三 AST | 演示四 MCP | "知识建设让 AI 理解代码——MCP 让 AI 连接外部系统" |
| 工具 → 流程 | 演示四 MCP | 演示五 Skill | "MCP 管数据拉取——Skill 管分流流程" |
| MCP+Skill → 代码审查 | 演示四+五 | 演示五 Step 0 | "AI 不只拉数据——还回到本地代码做验证" |
| Skill → 闭环 | 演示五 Skill | 演示六 迭代 | "分流出 TRUE POSITIVE——然后 AI 进入修复闭环" |
| 闭环 → 安全 | 演示六 迭代 | 演示七 安全 | "AI 能自主修复——但修复的安全性谁来保证？" |
